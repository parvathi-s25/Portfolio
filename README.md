# Parvathi S | Portfolio

A modern, dark-themed AI & Data Science portfolio website built with semantic HTML5, CSS3 (Glassmorphism), and Vanilla JavaScript.

## 🚀 Live Demo
[https://parvathi-portfolio.vercel.app](https://parvathi-portfolio.vercel.app)

## 🛠️ Features
- **Modern UI**: Dark-themed with neon blue accents and glassmorphism.
- **Fully Responsive**: Optimized for all devices (Mobile, Tablet, Desktop).
- **Animations**: Scroll-reveal animations and interactive hover effects.
- **QR Code System**: Built-in QR generator for easy sharing.
- **Dynamic Active Nav**: Highlights current section on scroll.

## 📂 Project Structure
- `index.html`: Main portfolio content.
- `style.css`: Design tokens and layout.
- `script.js`: Interactivity and animations.
- `qr.html`: QR code generation page.
- `vercel.json`: Deployment configuration.

## 🚀 Deployment Instructions

### Method 1: Vercel CLI (Fastest)
1. Install Vercel CLI: `npm i -g vercel`
2. Run: `vercel` (follow the prompts to deploy to your personal account)

### Method 2: GitHub Dashboard
1. Push this project to a GitHub repository.
2. Go to [vercel.com](https://vercel.com).
3. Click "Add New" > "Project".
4. Import your repository.
5. Vercel will automatically detect the static project and deploy it.

## 📱 QR Testing
1. Navigate to `qr.html` (or click the QR icon in the navbar).
2. Scan the generated QR code with your phone.
3. It will redirect to the live portfolio URL.
4. Use the "Download PNG" button to save the QR code for posters or business cards.
